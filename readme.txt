== Description ==

Adds a date range field to CMB2

== Installation ==

= Manual Installation =

1. Upload the entire `/wds-cmb2-date-range-field` directory to the `/wp-content/plugins/` directory.
2. Activate WDS CMB2 Date Range Field through the 'Plugins' menu in WordPress.
